## 安装说明

Windows 用户下载 `.exe` 后直接运行安装。

macOS 当前安装包未做 Apple 公证。如果提示无法验证开发者，请在访达中找到应用，右键选择“打开”，再点击“打开”。如果仍被拦截，请进入“系统设置 → 隐私与安全性”，允许打开本应用。

## 使用顺序

1. 打开软件后先进入“网络配置”，选择扫描到的本机连接方式并测试成功。
2. 回到首页完成网页登录。
3. 在软件里选择或创建 Key。
4. 进入“软件配置中心”，选择 Codex / Claude Code / Claude Desktop，点击“导入并启用当前 Key”。
5. 只有 Codex 需要进入“聊天记录恢复”，先安全扫描，再按提示恢复。
